## 图解风格

所有图解使用 ASCII 艺术，确保在终端和 Markdown 中都能正确显示。

### 示例：栈与堆

```
┌─────────────────────────────────────────┐
│              内存布局                     │
├─────────────────────────────────────────┤
│                                         │
│  栈内存 (Stack)                          │
│  ┌──────────┐                           │
│  │  x = 5   │ ← 快速访问                  │
│  │  y = 10  │   固定大小                  │
│  │  z = 15  │   自动释放                  │
│  └──────────┘                           │
│                                         │
│  堆内存 (Heap)                           │
│  ┌─────────────────┐                   │
│  │ String "hello"  │ ← 动态分配          │
│  │ [h,e,l,l,o]     │   可变大小          │
│  │ ptr ────────────┼─┐ 手动管理         │
│  └─────────────────┘ │                 │
│                        │                │
│  栈指向堆:             │                │
│  ┌──────────┐        │                │
│  │ s1 (ptr) │────────┘                │
│  │ len = 5  │                          │
│  │ cap = 5  │                          │
│  └──────────┘                          │
└─────────────────────────────────────────┘
```

### 示例：所有权移动

```
时间线：String 的所有权移动

T0: let s1 = String::from("hello");
┌────────────────────┐
│  栈                │
│  ┌──────────┐      │
│  │ s1       │      │
│  │ ptr ─────┼──┐   │
│  │ len = 5  │  │   │
│  │ cap = 5  │  │   │
│  └──────────┘  │   │
└────────────────┼───┘
                 │
┌────────────────┼────┐
│  堆             │    │
│  ┌──────────┐   │    │
│  │ [hello]  │◄──┘    │
│  └──────────┘        │
└──────────────────────┘

T1: let s2 = s1;
┌────────────────────┐
│  栈                │
│  ┌──────────┐      │
│  │ s1 ❌     │      │  ← s1 失效
│  └──────────┘      │
│  ┌──────────┐      │
│  │ s2       │      │  ← s2 获得所有权
│  │ ptr ─────┼──┐   │
│  │ len = 5  │  │   │
│  │ cap = 5  │  │   │
│  └──────────┘  │   │
└────────────────┼───┘
                 │
┌────────────────┼────┐
│  堆             │    │
│  ┌──────────┐   │    │
│  │ [hello]  │◄──┘    │
│  └──────────┘        │
└──────────────────────┘
```
