## 切片概念速查

```
┌─────────────────────────────────────────────────────────────┐
│                    切片（Slice）                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  切片 = 对集合中一段连续元素的引用                            │
│                                                             │
│  ┌─────────────────────────────────────┐                   │
│  │         胡指针结构                   │                   │
│  ├─────────────────────────────────────┤                   │
│  │  • ptr    → 指向起始位置             │                   │
│  │  • len    → 元素数量                 │                   │
│  │  • 不拥有数据，只是引用               │                   │
│  └─────────────────────────────────────┘                   │
│                                                             │
│  类型表示：                                                  │
│  &[T]   → 数组/Vec 切片                                     │
│  &str   → 字符串切片                                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 类型 | 说明 | 示例 |
|------|------|------|
| `&str` | 字符串切片 | `&s[0..5]` |
| `&[T]` | 数组/Vec切片 | `&arr[1..3]` |
| `&mut [T]` | 可变切片 | `&mut v[0..2]` |
