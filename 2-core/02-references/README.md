## 借用规则速查

```
┌─────────────────────────────────────────────────────────────┐
│                    借用规则                                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 可以有多个不可变引用                                     │
│     let r1 = &s;                                            │
│     let r2 = &s;  ✅ 同时存在                               │
│                                                             │
│  2. 可以有一个可变引用                                       │
│     let r1 = &mut s;                                        │
│     let r2 = &mut s;  ❌ 不能同时存在                       │
│                                                             │
│  3. 可变引用与不可变引用不能同时存在                         │
│     let r1 = &s;                                            │
│     let r2 = &mut s;  ❌ 不能同时存在                       │
│                                                             │
│  核心原则：引用的有效范围不能超过所有者                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 类型 | 说明 | 示例 |
|------|------|------|
| `&T` | 不可变引用 | `let r = &x;` |
| `&mut T` | 可变引用 | `let r = &mut x;` |
| `*` | 解引用 | `*r` |
| 借用 | 临时使用权 | 不转移所有权 |
