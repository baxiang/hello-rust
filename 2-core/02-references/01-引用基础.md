## дёәд»Җд№ҲйңҖиҰҒеј•з”Ёпјҹ

### жүҖжңүжқғзҡ„еұҖйҷҗжҖ§

**жҰӮеҝөеҗҚз§°пјҡ** еј•з”ЁпјҲReferenceпјүе…Ғи®ёеҖҹз”ЁеҖјиҖҢдёҚиҺ·еҸ–жүҖжңүжқғгҖӮ

```
иҜӯжі•з»“жһ„пјҡ
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ  &T     вҶ’ дёҚеҸҜеҸҳеј•з”ЁпјҲеҸӘиҜ»пјү           в”Ӯ
в”Ӯ  &mut T вҶ’ еҸҜеҸҳеј•з”ЁпјҲеҸҜиҜ»еҶҷпјү           в”Ӯ
в”Ӯ                                       в”Ӯ
в”Ӯ  let r = &s;      вҶ’ еҖҹз”Ё s            в”Ӯ
в”Ӯ  let r = &mut s;  вҶ’ еҸҜеҸҳеҖҹз”Ё s        в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

### жңҖз®ҖзӨәдҫӢ

```rust
fn main() {
    let s = String::from("hello");
    let r = &s;  // еҖҹз”ЁпјҢдёҚз§»еҠЁжүҖжңүжқғ
    println!("{}", r);  // йҖҡиҝҮеј•з”Ёи®ҝй—®
    println!("{}", s);  // s д»Қз„¶жңүж•Ҳ
}
```

### иҜҰз»ҶзӨәдҫӢ

```rust
// й—®йўҳпјҡеҮҪж•°иҺ·еҸ–жүҖжңүжқғеҗҺпјҢеҺҹеҸҳйҮҸдёҚиғҪдҪҝз”Ё
fn main() {
    let s = String::from("hello");
    let len = string_length(s);  // s зҡ„жүҖжңүжқғиў«з§»еҠЁ

    // println!("{}", s);  // вқҢ й”ҷиҜҜпјҡs е·Іж— ж•Ҳ
    println!("й•ҝеәҰпјҡ{}", len);
}

fn string_length(s: String) -> usize {
    s.len()
}
```

### еј•з”Ёзҡ„и§ЈеҶіж–№жЎҲ

```rust
// дҪҝз”Ёеј•з”ЁпјҡдёҚиҪ¬з§»жүҖжңүжқғ
fn main() {
    let s = String::from("hello");
    let len = string_length(&s);  // еҸӘеҖҹз”ЁпјҢдёҚз§»еҠЁ

    println!("'{}' зҡ„й•ҝеәҰжҳҜ {}", s, len);  // вң… s д»Қз„¶жңүж•Ҳ
}

fn string_length(s: &String) -> usize {
    s.len()
}
```

## дёҚеҸҜеҸҳеј•з”ЁпјҲ&Tпјү

### еј•з”ЁиҜӯжі•

**жҰӮеҝөеҗҚз§°пјҡ** `&T` еҲӣе»әдёҚеҸҜеҸҳеј•з”ЁпјҲеҸӘиҜ»еҖҹз”ЁпјүгҖӮ

### жңҖз®ҖзӨәдҫӢ

```rust
fn main() {
    let s = String::from("hello");
    let len = calculate_length(&s);
    println!("'{}' й•ҝеәҰ {}", s, len);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}
```

### иҜҰз»ҶзӨәдҫӢ

```rust
fn main() {
    let s = String::from("hello");

    // еҲӣе»әеј•з”ЁпјҡдҪҝз”Ё &
    let r1 = &s;
    let r2 = &s;

    // дҪҝз”Ёеј•з”Ё
    println!("r1: {}", r1);
    println!("r2: {}", r2);

    // еҺҹеҸҳйҮҸд»Қз„¶жңүж•Ҳ
    println!("s: {}", s);
}
```

    println!("'{}' зҡ„й•ҝеәҰжҳҜ {}", s, len);

### еӨҡдёӘдёҚеҸҜеҸҳеј•з”Ё

```rust
fn main() {
    let s = String::from("hello");

    // еҸҜд»ҘеҗҢж—¶жңүеӨҡдёӘдёҚеҸҜеҸҳеј•з”Ё
    let r1 = &s;
    let r2 = &s;
    let r3 = &s;

    println!("r1: {}, r2: {}, r3: {}", r1, r2, r3);

    // еҺҹеҸҳйҮҸд№ҹеҸҜд»ҘдҪҝз”Ё
    println!("s: {}", s);
}
```
