## 结构体类型速查

```
┌─────────────────────────────────────────────────────────────┐
│                    结构体类型                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  普通结构体                                                  │
│  struct Person {                                            │
│      name: String,                                          │
│      age: u32,                                              │
│  }                                                          │
│                                                             │
│  元组结构体（字段无名）                                       │
│  struct Point(i32, i32);                                    │
│                                                             │
│  单元结构体（无字段）                                         │
│  struct Unit;                                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 概念 | 说明 | 示例 |
|------|------|------|
| `struct` | 定义结构体 | `struct User { name: String }` |
| 方法 | 实例调用 | `user.get_name()` |
| 关联函数 | 类型调用 | `User::new()` |
| `impl` | 实现块 | `impl User { ... }` |
