## 学习路线

### 前置要求

- 已完成第 1-28 章所有内容
- 熟悉 Rust 核心概念：所有权、Trait、错误处理
- 了解并发编程基础
- 掌握基本的 Web 开发概念

### 学习建议

1. **按序学习**：三个章节有递进关系，建议按顺序学习
2. **实践为主**：每章包含大量实战案例，务必动手实践
3. **关注生态**：现代实践涉及大量第三方 crate，理解其设计理念
4. **持续更新**：这部分内容会随生态系统演进，关注最新动态
