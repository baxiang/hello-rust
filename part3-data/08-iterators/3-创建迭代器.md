## 19.3 创建迭代器

### iter()、iter_mut()、into_iter()

```
┌─────────────────────────────────────────────────────┐
│           三种创建迭代器的方式                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│  iter() - 不可变借用迭代器                          │
│  ├── 类型：Iterator<Item = &T>                      │
│  ├── 不获取所有权                                   │
│  ├── 不能修改元素                                   │
│  └── 原集合迭代后仍可用                             │
│                                                     │
│  iter_mut() - 可变借用迭代器                        │
│  ├── 类型：Iterator<Item = &mut T>                  │
│  ├── 不获取所有权                                   │
│  ├── 可以修改元素                                   │
│  └── 迭代期间原集合不可用                           │
│                                                     │
│  into_iter() - 所有权迭代器                         │
│  ├── 类型：Iterator<Item = T>                       │
│  ├── 获取所有权                                     │
│  ├── 消耗元素                                       │
│  └── 原集合迭代后不可用                             │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 三种方式对比

```rust
fn main() {
    let mut v = vec![1, 2, 3];

    // ========== iter() - 不可变借用 ==========
    println!("\n使用 iter():");
    for item in v.iter() {
        println!("  {}", item);  // &i32
    }
    println!("v 仍然可用：{:?}", v);

    // ========== iter_mut() - 可变借用 ==========
    println!("\n使用 iter_mut():");
    for item in v.iter_mut() {
        *item *= 2;  // 修改元素
        println!("  {}", item);
    }
    println!("v 修改后：{:?}", v);

    // ========== into_iter() - 所有权 ==========
    println!("\n使用 into_iter():");
    for item in v.into_iter() {
        println!("  {}", item);  // i32（所有权）
    }
    // v 已移动，不能再使用
    // println!("{:?}", v);  // ❌ 错误
}
```

### for 循环的脱糖

```rust
// for 循环实际上使用迭代器
for item in collection {
    println!("{}", item);
}

// 等价于
let mut iter = collection.into_iter();
while let Some(item) = iter {
    println!("{}", item);
}
```
