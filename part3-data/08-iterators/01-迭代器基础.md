## 19.2 Iterator Trait

### Trait 定义

```rust
// 标准库中的定义
pub trait Iterator {
    type Item;  // 关联类型：迭代的元素类型

    // 唯一必需的方法
    fn next(&mut self) -> Option<Self::Item>;

    // 大量默认实现的方法（适配器）
    fn map<F, B>(self, f: F) -> Map<Self, F> { ... }
    fn filter<P>(self, predicate: P) -> Filter<Self, P> { ... }
    fn collect<B>(self) -> B { ... }
    // ... 等等
}
```

### next() 方法详解

```
next() 的工作流程：

┌─────────────────────────────────────────────────────┐
│                 迭代器状态机                         │
├─────────────────────────────────────────────────────┤
│                                                     │
│  每次调用 next()：                                   │
│                                                     │
│  有下一个元素  ──────→  Some(element)               │
│                      │                              │
│  没有元素了   ──────→  None（迭代结束）              │
│                                                     │
│  示意图：                                            │
│  ┌─────┬─────┬─────┬─────┬─────┐                   │
│  │  A  │  B  │  C  │  D  │     │                   │
│  └─────┴─────┴─────┴─────┴─────┘                   │
│     ↑                                               │
│     当前位置                                        │
│                                                     │
│  next() → Some(A)，位置前进                         │
│  next() → Some(B)，位置前进                         │
│  next() → Some(C)，位置前进                         │
│  next() → Some(D)，位置前进                         │
│  next() → None，结束                                │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 手动迭代示例

```rust
fn main() {
    let v = vec![1, 2, 3];
    let mut iter = v.iter();

    // 手动调用 next()
    println!("{:?}", iter.next());  // Some(&1)
    println!("{:?}", iter.next());  // Some(&2)
    println!("{:?}", iter.next());  // Some(&3)
    println!("{:?}", iter.next());  // None
    println!("{:?}", iter.next());  // None（已经结束）
}
```

## 19.3 创建迭代器

### iter()、iter_mut()、into_iter()

```
┌─────────────────────────────────────────────────────┐
│           三种创建迭代器的方式                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│  iter() - 不可变借用迭代器                          │
│  ├── 类型：Iterator<Item = &T>                      │
│  ├── 不获取所有权                                   │
│  ├── 不能修改元素                                   │
│  └── 原集合迭代后仍可用                             │
│                                                     │
│  iter_mut() - 可变借用迭代器                        │
│  ├── 类型：Iterator<Item = &mut T>                  │
│  ├── 不获取所有权                                   │
│  ├── 可以修改元素                                   │
│  └── 迭代期间原集合不可用                           │
│                                                     │
│  into_iter() - 所有权迭代器                         │
│  ├── 类型：Iterator<Item = T>                       │
│  ├── 获取所有权                                     │
│  ├── 消耗元素                                       │
│  └── 原集合迭代后不可用                             │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 三种方式对比

```rust
fn main() {
    let mut v = vec![1, 2, 3];

    // ========== iter() - 不可变借用 ==========
    println!("\n使用 iter():");
    for item in v.iter() {
        println!("  {}", item);  // &i32
    }
    println!("v 仍然可用：{:?}", v);

    // ========== iter_mut() - 可变借用 ==========
    println!("\n使用 iter_mut():");
    for item in v.iter_mut() {
        *item *= 2;  // 修改元素
        println!("  {}", item);
    }
    println!("v 修改后：{:?}", v);

    // ========== into_iter() - 所有权 ==========
    println!("\n使用 into_iter():");
    for item in v.into_iter() {
        println!("  {}", item);  // i32（所有权）
    }
    // v 已移动，不能再使用
    // println!("{:?}", v);  // ❌ 错误
}
```

### for 循环的脱糖

```rust
// for 循环实际上使用迭代器
for item in collection {
    println!("{}", item);
}

// 等价于
let mut iter = collection.into_iter();
while let Some(item) = iter {
    println!("{}", item);
}
```
