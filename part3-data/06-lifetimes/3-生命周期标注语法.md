## 17.3 生命周期标注语法

### 基本语法

```rust
// 生命周期参数以撇号开头，通常小写
// 常见命名：'a, 'b, 'c, ...

// 带生命周期的引用
let r: &'a i32;           // 不可变引用
let r: &'a mut i32;       // 可变引用

// 多个生命周期
let r: &'a &'b i32;       // 引用的引用
let r: &'a mut &'b i32;   // 可变引用

// 生命周期参数的位置
fn function<'a>(param: &'a i32) { }
struct Struct<'a> { field: &'a str }
enum Enum<'a> { Variant(&'a str) }
```

### 生命周期的含义

```rust
// &'a i32 的含义：
// "这个引用在生命周期 'a 内有效"

// 示例：
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    //            ↑              ↑           ↑
    //            │              │           │
    //         x 的生命周期   y 的生命周期  返回的生命周期
    //
    // 'a 的含义：返回的引用生命周期
    //           不能超过 x 和 y 的生命周期
    if x.len() > y.len() {
        x
    } else {
        y
    }
}
```

### 生命周期标注的位置

```rust
// 1. 函数参数和返回值
fn foo<'a>(x: &'a str) -> &'a str {
    x
}

// 2. 结构体字段
struct Bar<'a> {
    name: &'a str,
}

// 3. 枚举变体
enum Baz<'a> {
    Name(&'a str),
    Value { data: &'a str },
}

// 4. impl 块
impl<'a> Bar<'a> {
    fn get(&self) -> &str {
        self.name
    }
}
```
