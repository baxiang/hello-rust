## 17.2 生命周期的基本概念

### 什么是生命周期

```
生命周期（Lifetime）是代码中某个引用有效的范围。

┌─────────────────────────────────────────────────────┐
│              生命周期图解                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  示例代码：                                          │
│  {                        作用域开始                 │
│      let x = 5;           x 的生命周期开始 'x        │
│      {                                               │
│          let r = &x;      r 的生命周期开始 'r        │
│          println!("{}", r);                          │
│      }                    r 的生命周期结束           │
│  }                        x 的生命周期结束，x 释放    │
│                                                     │
│  生命周期关系：                                      │
│  'r ⊆ 'x  （r 的生命周期必须包含在 x 的生命周期内）   │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 作用域嵌套与生命周期

```rust
fn main() {
    let x = 10;              // 'x 开始
    {
        let r = &x;          // 'r 开始，'r ⊆ 'x
        println!("{}", r);
    }                        // 'r 结束
    {
        let s = &x;          // 's 开始，'s ⊆ 'x
        println!("{}", s);
    }                        // 's 结束
}                            // 'x 结束，x 被释放
```

### 编译器如何检查生命周期

```
编译器在编译时进行借用检查：

1. 跟踪每个引用的创建和销毁
2. 验证引用不会超出其指向数据的生命周期
3. 确保借用规则不被违反

示例分析：
┌───────────────────────────────────────────────┐
│ fn main() {                                   │
│     let x = 5;          // 'x 开始            │
│     let r = &x;         // 'r 开始，绑定到'x  │
│     println!("{}", r);  // 使用 r，有效       │
│ }                     // 'r 结束，然后'x 结束  │
│                                               │
│ 检查通过：'r ⊆ 'x                            │
└───────────────────────────────────────────────┘

错误示例：
┌───────────────────────────────────────────────┐
│ fn dangle() -> &i32 {                         │
│     let x = 5;          // 'x 开始            │
│     &x                // 返回&x，'x 应该继续  │
│ }                     // 'x 结束，x 被释放！  │
│                                               │
│ 检查失败： 返回的引用超出了'x                 │
└───────────────────────────────────────────────┘
```
