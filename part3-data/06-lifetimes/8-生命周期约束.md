## 17.8 生命周期约束

### 生命周期之间的包含关系

```rust
// 'a: 'b 表示 'a 至少和 'b 一样长（'a 包含 'b）

// 示例：返回较短的生命周期
fn shortest<'a, 'b>(x: &'a str, y: &'b str) -> &'a str
where
    'a: 'b,  // 'a 包含 'b
{
    x
}

// 含义：
// • 'a 至少和 'b 一样长
// • 返回'x 的引用是安全的，因为'a 更长
```

### 多个生命周期约束

```rust
// 多约束示例
fn process<'a, 'b, 'c>(
    x: &'a str,
    y: &'b str,
    z: &'c str,
) -> &'a str
where
    'a: 'b + 'c,  // 'a 包含 'b 和 'c
{
    x
}
```
