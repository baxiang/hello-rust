## 15.2 泛型函数

### 基本语法

```rust
// 泛型函数语法
// fn 函数名<泛型参数列表>(参数) -> 返回值 { 函数体 }

// 单个泛型参数
fn print_item<T>(item: T) {
    println!("item: {:?}", item);
}

// 多个泛型参数
fn print_pair<T, U>(t: T, u: U) {
    println!("t: {:?}, u: {:?}", t, u);
}

fn main() {
    // 类型推断：编译器自动推断 T 的类型
    print_item(42);           // T = i32
    print_item("hello");      // T = &str
    print_item(3.14);         // T = f64

    // 显式指定类型
    print_item::<i32>(100);
    print_item::<String>(String::from("test"));

    // 多个泛型参数
    print_pair(42, "hello");        // T=i32, U=&str
    print_pair(3.14, true);         // T=f64, U=bool
    print_pair("a", vec![1, 2]);    // T=&str, U=Vec<i32>
}
```

### 泛型参数的命名约定

```rust
// 泛型参数通常使用大写字母，简短有意义
// 常见约定：
// T - Type（类型）
// U, V, W - 其他类型
// K, V - Key, Value（键值对）
// R - Return type（返回类型）
// E - Error（错误类型）
// A, B - 任意类型

// 示例
fn identity<T>(x: T) -> T { x }
fn convert<A, B>(x: A) -> B { /* ... */ }
fn get<K, V>(map: &HashMap<K, V>, key: &K) -> Option<&V> { /* ... */ }
```

### 泛型与返回值

```rust
// 返回泛型类型
fn first_element<T>(vec: Vec<T>) -> Option<T> {
    vec.into_iter().next()
}

// 返回引用
fn get_first<T>(slice: &[T]) -> Option<&T> {
    slice.first()
}

// 多个返回值
fn swap<T>(a: T, b: T) -> (T, T) {
    (b, a)
}

fn main() {
    let vec = vec![1, 2, 3];
    println!("{:?}", first_element(vec));

    let arr = [10, 20, 30];
    println!("{:?}", get_first(&arr));

    let (x, y) = swap(1, 2);
    println!("x={}, y={}", x, y);
}
```
