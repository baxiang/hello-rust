## 15.11 小结

### 泛型核心概念

| 概念 | 说明 |
|------|------|
| 泛型函数 | `fn foo<T>(x: T) -> T` |
| 泛型结构体 | `struct Foo<T> { x: T }` |
| 泛型枚举 | `enum Foo<T> { Bar(T) }` |
| Trait 约束 | `T: Trait1 + Trait2` |
| where 子句 | 复杂的约束条件 |
| 单态化 | 编译时生成专用代码 |

### 常用约束

| Trait | 用途 |
|-------|------|
| `Display` | 格式化输出 |
| `Debug` | 调试输出 |
| `Clone` | 克隆 |
| `Copy` | 复制 |
| `PartialOrd` | 部分排序 |
| `PartialEq` | 相等比较 |
| `Eq` | 完全相等 |
| `Ord` | 完全排序 |

### 最佳实践

1. **在函数/方法上添加约束**，而不是结构体定义
2. **使用 where 子句**处理复杂约束
3. **保持简单**：不要过度泛化
4. **考虑性能**：单态化可能导致代码膨胀
5. **提供有用的错误信息**：使用 expect 而不是 unwrap
