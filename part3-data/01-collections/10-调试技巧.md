## 12.10 调试技巧

### 打印调试信息

```rust
fn main() {
    let v = vec![1, 2, 3, 4, 5];

    // 基本调试输出
    println!("{:?}", v);

    // 美化格式
    println!("{:#?}", v);

    // 使用 dbg! 宏
    let result = dbg!(v.len()) * 2;

    // dbg! 返回值的引用，可以用于链式调用
    let v = vec![1, 2, 3];
    let sum: i32 = dbg!(v).iter().sum();
}
```

### 检查容量变化

```rust
fn main() {
    let mut v: Vec<i32> = Vec::with_capacity(4);

    println!("初始：len={}, cap={}", v.len(), v.capacity());

    for i in 0..10 {
        v.push(i);
        println!("push({}): len={}, cap={}", i, v.len(), v.capacity());
    }
}
```
