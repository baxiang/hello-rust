## 13.13 调试技巧

### 打印 HashMap

```rust
use std::collections::HashMap;

fn main() {
    let mut map = HashMap::new();
    map.insert("a", 1);
    map.insert("b", 2);

    // 基本输出
    println!("{:?}", map);

    // 美化格式
    println!("{:#?}", map);
}
```

### 调试容量和负载

```rust
use std::collections::HashMap;

fn main() {
    let mut map: HashMap<i32, i32> = HashMap::with_capacity(10);

    println!("初始容量：{}", map.capacity());

    for i in 0..20 {
        map.insert(i, i * 2);
        println!("插入 {} 后：len={}, cap={}", i, map.len(), map.capacity());
    }
}
```
