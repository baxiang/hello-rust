## 11.2 定义和创建枚举

### 基本语法

```rust
// 定义枚举
enum Direction {
    North,
    South,
    East,
    West,
}

fn main() {
    // 创建枚举实例
    let dir = Direction::North;

    // 使用 match 处理
    match dir {
        Direction::North => println!("向北"),
        Direction::South => println!("向南"),
        Direction::East => println!("向东"),
        Direction::West => println!("向西"),
    }
}
```

### 枚举的内存表示

```
enum Direction {
    North,  // 0
    South,  // 1
    East,   // 2
    West,   // 3
}

内存布局：
┌─────┐
│  0  │  ← 用一个整数表示
└─────┘

大小：1 字节（可以用 u8 存储 4 个值）
```

### 带数据的枚举变体

```rust
enum Message {
    // 无数据
    Quit,

    // 命名字段（类似结构体）
    Move { x: i32, y: i32 },

    // 单个值
    Write(String),

    // 多个值（类似元组）
    ChangeColor(i32, i32, i32),
}

fn main() {
    // 创建不同变体
    let m1 = Message::Quit;
    let m2 = Message::Move { x: 10, y: 20 };
    let m3 = Message::Write(String::from("hello"));
    let m4 = Message::ChangeColor(255, 0, 0);

    // 每个变体可以携带不同类型和大小的数据
}
```

### 带数据枚举的内存布局

```
enum Message {
    Quit,                      // 无数据
    Move { x: i32, y: i32 },  // 8 字节
    Write(String),             // 24 字节
    ChangeColor(i32, i32, i32), // 12 字节
}

内存布局：
┌─────┬─────────────────────────────┐
│标签 │     数据（最大变体）         │
│2bit │        24 字节               │
└─────┴─────────────────────────────┘

• 标签（tag）：标识是哪个变体
• 数据区：足以容纳最大变体
• 总大小：约 32 字节
```
