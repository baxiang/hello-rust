## 枚举概念速查

```
┌─────────────────────────────────────────────────────────────┐
│                    枚举（Enum）                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  枚举 = 一组命名的变体（Variant）                            │
│                                                             │
│  enum Direction {                                           │
│      North,  // 无数据变体                                   │
│      East,                                                   │
│      South,                                                  │
│      West,                                                   │
│  }                                                          │
│                                                             │
│  enum Message {                                             │
│      Quit,                        // 无数据                  │
│      Move { x: i32, y: i32 },     // 像结构体                │
│      Write(String),               // 单个值                  │
│      ChangeColor(i32, i32, i32),  // 多个值                  │
│  }                                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 概念 | 说明 | 示例 |
|------|------|------|
| `enum` | 定义枚举 | `enum Direction { North, South }` |
| `match` | 模式匹配 | `match dir { North => ... }` |
| `if let` | 简化匹配 | `if let Some(x) = opt { }` |
| `Option` | 可选值 | `Some(value)` / `None` |
| `Result` | 错误处理 | `Ok(value)` / `Err(e)` |
