## 9.10 安全特性

### 边界检查

```rust
fn main() {
    let numbers = [1, 2, 3, 4, 5];

    // 切片会进行边界检查
    // 如果范围无效，会 panic

    // ❌ 这会 panic
    // let bad = &numbers[10..15];

    // ✅ 安全做法：使用 get
    match numbers.get(10..) {
        Some(slice) => println!("{:?}", slice),
        None => println!("索引超出范围"),
    }

    // ✅ 或者先检查
    if numbers.len() >= 10 {
        let slice = &numbers[5..10];
        println!("{:?}", slice);
    } else {
        println!("数组太短");
    }
}
```

### UTF-8 字符边界

```rust
fn main() {
    let s = String::from("你好世界");

    // ❌ 错误：在字符边界外切片
    // &s[0..1];  // panic! 汉字是 3 字节，0..1 不是有效边界

    // ✅ 正确：使用字符迭代
    for c in s.chars() {
        println!("{}", c);
    }

    // ✅ 正确：在字符边界切片
    // 每个汉字 3 字节
    let first = &s[0..3];   // "你"
    let second = &s[3..6];  // "好"
    println!("{} {}", first, second);

    // ✅ 安全切片：使用 get
    match s.get(0..3) {
        Some(slice) => println!("{}", slice),
        None => println!("无效的字符边界"),
    }
}
```

### panic 信息

```
当切片边界无效时：

thread 'main' panicked at 'byte index 1 is not a char boundary; it is inside '' (bytes 0..3) of ``'

或者：

thread 'main' panicked at 'range end index 100 out of range for slice of length 5'
```
