## 8.7 解引用操作符（*）

### 基本用法

```rust
fn main() {
    let x = 5;
    let y = &x;  // y 是引用

    assert_eq!(5, x);
    assert_eq!(5, *y);  // *y 解引用，获取值
}
```

### 内存图解

```
栈：
┌──────┐
│  x=5 │
└──────┘
   ↑
   │
┌──────┐
│  y   │  → 指向 x
└──────┘

*y = 5（跟随指针获取值）
```

### 自动解引用（Deref Coercion）

```rust
fn main() {
    let mut s = String::from("hello");
    let r = &mut s;

    // Rust 会自动解引用
    r.push_str(", world");

    // 等价于显式解引用
    (*r).push_str("!");

    println!("{}", s);
}
```

### 何时需要显式解引用

```rust
fn main() {
    let x = 5;
    let y = &x;

    // 需要显式解引用的情况
    let z = *y + 1;  // z = 6

    // 不需要解引用的情况
    println!("{}", y);  // Rust 自动解引用
    assert_eq!(5, y);   // Rust 自动解引用
}
```
