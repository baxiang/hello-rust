## 8.11 调试技巧

### 打印借用信息

```rust
fn main() {
    let s = String::from("hello");
    let r = &s;

    println!("s 的地址：{:p}", &s);
    println!("r 的地址：{:p}", &r);
    println!("r 指向的地址：{:p}", r.as_ptr());
}
```

### 跟踪借用

```rust
fn main() {
    let mut s = String::from("hello");
    println!("初始：{}", s);

    {
        let r = &mut s;
        println!("借用中：{}", r);
        r.push_str(" world");
    }  // 借用结束

    println!("借用结束后：{}", s);
}
```
