# 所有权常见错误

> 避免这些常见陷阱


## 错误 1：使用已移动的值

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1;

    // println!("{}", s1);  // ❌ 错误：使用了已移动的值
    println!("{}", s2);  // ✅ 正确
}
```

**错误信息：**
```
error[E0382]: use of moved value: `s1`
 --> src/main.rs:5:20
  |
3 |     let s1 = String::from("hello");
  |         -- move occurs because `s1` has type `String`
4 |     let s2 = s1;
  |              -- value moved here
5 |     println!("{}", s1);
  |                    ^^ value used here after move
```


## 错误 2：函数参数移动后继续使用

```rust
fn print_string(s: String) {
    println!("{}", s);
}

fn main() {
    let s = String::from("hello");
    print_string(s);

    // println!("{}", s);  // ❌ 错误：s 已移动
}
```

### 解决方案

```rust
// 方案 1：传递引用
fn print_string(s: &String) {
    println!("{}", s);
}

fn main() {
    let s = String::from("hello");
    print_string(&s);
    println!("{}", s);  // ✅ s 仍然有效
}

// 方案 2：接受 &str（更佳）
fn print_string(s: &str) {
    println!("{}", s);
}

fn main() {
    let s = String::from("hello");
    print_string(&s);
    print_string("literal");
}

// 方案 3：使用 clone
fn main() {
    let s = String::from("hello");
    print_string(s.clone());
    println!("{}", s);  // ✅ s 仍然有效
}
```


## 错误 3：在循环中移动

```rust
fn main() {
    let s = String::from("hello");

    // ❌ 错误：试图在循环中多次使用 s
    // for _ in 0..3 {
    //     print_string(s);  // 第一次后 s 就无效了
    // }
}
```

### 解决方案

```rust
// 方案 1：传递引用
fn main() {
    let s = String::from("hello");

    for _ in 0..3 {
        print_string(&s);
    }
}

fn print_string(s: &str) {
    println!("{}", s);
}

// 方案 2：使用 clone
fn main() {
    let s = String::from("hello");

    for _ in 0..3 {
        print_string(s.clone());
    }
    println!("{}", s);
}
```


## 错误 4：返回局部变量的引用

```rust
// ❌ 错误：试图返回局部变量的引用
// fn get_string() -> &String {
//     let s = String::from("hello");
//     &s  // s 在函数结束时被释放
// }

// ✅ 正确：返回 String（转移所有权）
fn get_string() -> String {
    String::from("hello")
}

fn main() {
    let s = get_string();
    println!("{}", s);
}
```


## 错误速查表

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 使用已移动的值 | 所有权已转移 | 使用引用或 clone |
| 函数后使用变量 | 参数移动所有权 | 传引用 &T |
| 循环中多次使用 | 第一次就移动了 | 传引用或 clone |
| 返回局部引用 | 作用域结束释放 | 返回所有权 |

# 所有权完整示例

## 示例 1：所有权传递链

```rust
fn main() {
    // 创建所有权链
    let s1 = String::from("start");
    let s2 = pass_to_function(s1);
    let s3 = modify_and_return(s2);

    println!("最终结果：{}", s3);
}

fn pass_to_function(s: String) -> String {
    println!("收到：{}", s);
    s  // 返回所有权
}

fn modify_and_return(s: String) -> String {
    let mut s = s;
    s.push_str(" - modified!");
    s  // 返回修改后的所有权
}
```

**输出：**
```
收到：start
最终结果：start - modified!
```


## 示例 2：字符串处理器

```rust
fn main() {
    let text = String::from("  Hello, Rust!  ");

    // 处理链
    let trimmed = trim_string(text);
    let upper = to_upper(trimmed);
    let greeted = add_greeting(upper);

    println!("结果：{}", greeted);
}

fn trim_string(s: String) -> String {
    s.trim().to_string()
}

fn to_upper(s: String) -> String {
    s.to_uppercase()
}

fn add_greeting(s: String) -> String {
    format!("Greeting: {}", s)
}
```

**输出：**
```
结果：Greeting: HELLO, RUST!
```


## 示例 3：数据转换管道

```rust
fn main() {
    let data = vec![1, 2, 3, 4, 5];

    // 所有权在管道中流动
    let doubled = double_all(data);
    let sum = sum_all(doubled);

    println!("总和：{}", sum);
}

fn double_all(numbers: Vec<i32>) -> Vec<i32> {
    numbers.into_iter().map(|n| n * 2).collect()
}

fn sum_all(numbers: Vec<i32>) -> i32 {
    numbers.into_iter().sum()
}
```

**输出：**
```
总和：30
```


## 示例 4：使用引用优化

```rust
fn main() {
    let s = String::from("hello");

    // 使用引用，避免所有权转移
    print_len(&s);
    print_content(&s);
    print_upper(&s);

    // s 仍然有效
    println!("原始字符串：{}", s);
}

fn print_len(s: &String) {
    println!("长度：{}", s.len());
}

fn print_content(s: &String) {
    println!("内容：{}", s);
}

fn print_upper(s: &String) {
    println!("大写：{}", s.to_uppercase());
}
```
