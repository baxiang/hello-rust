# 作用域详解

> 理解变量的生命周期

---

## 作用域基础

```rust
fn main() {
    // s 从这里开始有效
    let s = String::from("hello");

    println!("{}", s);  // 可以使用 s
}  // s 离开作用域，内存被自动释放
   // Rust 在这里隐式调用 drop(s)
```

---

## 作用域嵌套

```rust
fn main() {
    let outer = String::from("外层");

    {
        let inner = String::from("内层");
        println!("inner: {}", inner);  // ✅ 可以
        println!("outer: {}", outer);  // ✅ 也可以
    }  // inner 离开作用域，被释放

    println!("outer: {}", outer);  // ✅ outer 仍然有效
    // println!("inner: {}", inner);  // ❌ inner 已无效
}  // outer 离开作用域，被释放
```

---

## 作用域与内存释放时机

```rust
fn main() {
    let s1 = String::from("first");
    let s2 = String::from("second");
    let s3 = String::from("third");

    // s1, s2, s3 都在作用域中

    {
        let s4 = String::from("fourth");
        println!("s4: {}", s4);
    }  // s4 在这里被释放

    println!("s1: {}", s1);
    println!("s2: {}", s2);
    println!("s3: {}", s3);
}  // s3, s2, s1 按相反顺序被释放
   // 释放顺序：s3 → s2 → s1（后进先出）
```

---

## 作用域规则总结

| 规则 | 说明 |
|------|------|
| 进入作用域 | 变量开始有效 |
| 离开作用域 | 变量被释放 |
| 嵌套作用域 | 内层可访问外层，外层不能访问内层 |
| 释放顺序 | 后进先出（LIFO） |
