# 所有权常见错误

> 避免这些常见陷阱

---

## 错误 1：使用已移动的值

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1;

    // println!("{}", s1);  // ❌ 错误：使用了已移动的值
    println!("{}", s2);  // ✅ 正确
}
```

**错误信息：**
```
error[E0382]: use of moved value: `s1`
 --> src/main.rs:5:20
  |
3 |     let s1 = String::from("hello");
  |         -- move occurs because `s1` has type `String`
4 |     let s2 = s1;
  |              -- value moved here
5 |     println!("{}", s1);
  |                    ^^ value used here after move
```

## 错误 2：函数参数移动后继续使用

```rust
fn print_string(s: String) {
    println!("{}", s);
}

fn main() {
    let s = String::from("hello");
    print_string(s);

    // println!("{}", s);  // ❌ 错误：s 已移动
}
```

### 解决方案

```rust
// 方案 1：传递引用
fn print_string(s: &String) {
    println!("{}", s);
}

fn main() {
    let s = String::from("hello");
    print_string(&s);
    println!("{}", s);  // ✅ s 仍然有效
}

// 方案 2：接受 &str（更佳）
fn print_string(s: &str) {
    println!("{}", s);
}

fn main() {
    let s = String::from("hello");
    print_string(&s);
    print_string("literal");
}

// 方案 3：使用 clone
fn main() {
    let s = String::from("hello");
    print_string(s.clone());
    println!("{}", s);  // ✅ s 仍然有效
}
```

## 错误 3：在循环中移动

```rust
fn main() {
    let s = String::from("hello");

    // ❌ 错误：试图在循环中多次使用 s
    // for _ in 0..3 {
    //     print_string(s);  // 第一次后 s 就无效了
    // }
}
```

### 解决方案

```rust
// 方案 1：传递引用
fn main() {
    let s = String::from("hello");

    for _ in 0..3 {
        print_string(&s);
    }
}

fn print_string(s: &str) {
    println!("{}", s);
}

// 方案 2：使用 clone
fn main() {
    let s = String::from("hello");

    for _ in 0..3 {
        print_string(s.clone());
    }
    println!("{}", s);
}
```

## 错误 4：返回局部变量的引用

```rust
// ❌ 错误：试图返回局部变量的引用
// fn get_string() -> &String {
//     let s = String::from("hello");
//     &s  // s 在函数结束时被释放
// }

// ✅ 正确：返回 String（转移所有权）
fn get_string() -> String {
    String::from("hello")
}

fn main() {
    let s = get_string();
    println!("{}", s);
}
```

---

## 错误速查表

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 使用已移动的值 | 所有权已转移 | 使用引用或 clone |
| 函数后使用变量 | 参数移动所有权 | 传引用 &T |
| 循环中多次使用 | 第一次就移动了 | 传引用或 clone |
| 返回局部引用 | 作用域结束释放 | 返回所有权 |
