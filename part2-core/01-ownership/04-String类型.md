# String 类型详解

> String 与 &str 的区别

---

## String vs &str

这是 Rust 初学者最容易困惑的地方，我们详细对比：

```
┌─────────────────────────────────────────────────────┐
│           String vs &str 详细对比                   │
├─────────────────────────────────────────────────────┤
│                                                     │
│  String（拥有所有权的字符串）                        │
│  ├── 可增长、可修改                                  │
│  ├── 数据存储在堆上                                 │
│  ├── 大小：24 字节（指针 + 长度 + 容量）              │
│  ├── 创建：String::from("hello")                    │
│  └── 所有权转移时会影响性能（移动）                  │
│                                                     │
│  &str（字符串切片/视图）                            │
│  ├── 不可变（通常）                                  │
│  ├── 借用数据，不拥有所有权                         │
│  ├── 大小：16 字节（指针 + 长度）                     │
│  ├── 字面量："hello"（类型是 &'static str）          │
│  └── 传递时只复制指针，非常高效                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 内存布局可视化

```
字符串字面量 &str：
┌─────────────────────────────────────────┐
│           栈（Stack）                    │
│  ┌───────────────┬───────────────┐      │
│  │    指针       │     长度      │      │
│  │  (指向静态区) │    (5)        │      │
│  └───────┬───────┴───────────────┘      │
│          │                               │
│          ▼                               │
│  ┌─────────────────────────────────┐    │
│  │       静态区（编译时常量）        │    │
│  │  "hello\0"                      │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘

String（堆分配字符串）：
┌─────────────────────────────────────────┐
│           栈（Stack）                    │
│  ┌───────────┬───────────┬─────────┐   │
│  │   指针    │   长度    │  容量   │   │
│  │ (指向堆)  │   (5)     │  (10)   │   │
│  └─────┬─────┴───────────┴─────────┘   │
│        │                                │
│        ▼                                │
│  ┌─────────────────────────────────┐   │
│  │       堆（Heap）                 │   │
│  │  "hello"  │  空闲空间 (5)       │   │
│  └─────────────────────────────────┘   │
│                                         │
│  可以增长：push_str(" world")           │
└─────────────────────────────────────────┘
```

## 代码示例

```rust
fn main() {
    // &str - 字符串字面量
    let s1: &str = "hello";

    // String - 可增长字符串
    let s2: String = String::from("hello");
    let s3: String = "hello".to_string();
    let s4: String = "hello".to_owned();

    // 修改 String
    let mut s5 = String::from("hello");
    s5.push_str(" world!");
    println!("{}", s5);  // hello world!

    // &str 不能修改
    // s1.push_str(" world");  // ❌ 错误

    // 类型验证
    println!("s1 类型：{:?}", std::any::type_name_of_val(&s1));
    println!("s2 类型：{:?}", std::any::type_name_of_val(&s2));
}
```

---

## 选择建议

| 场景 | 推荐 |
|------|------|
| 函数参数 | `&str`（更通用） |
| 结构体字段 | `String`（拥有所有权） |
| 字符串字面量 | `&'static str` |
| 需要修改 | `String` |
