## 常见问题

- **为什么 Rust 需要所有权系统？**
  - 保证内存安全，无需 GC
  - 防止双重释放、悬垂指针
  - 编译时检查，零运行时开销

- **哪些类型实现了 Copy trait？**
  - 基本类型：i32, f64, bool, char
  - 元组：所有元素是 Copy
  - 数组：元素是 Copy
  - 非 Copy：String, Vec, Box 等

- **何时使用 Clone？**
  - 需要独立副本时
  - 保留原值所有权时
  - 大数据慎用，性能昂贵
