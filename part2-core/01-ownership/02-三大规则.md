# 所有权的三大规则

> Rust 内存管理的核心定律

---

## 所有权三定律

```
┌─────────────────────────────────────────────────────┐
│              所有权三定律                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  第一定律：每个值都有一个所有者                     │
│           Each value has an owner                  │
│                                                     │
│  第二定律：任一时刻只能有一个所有者                │
│           There can only be one owner at a time    │
│                                                     │
│  第三定律：当所有者离开作用域，值会被丢弃          │
│           When the owner goes out of scope,        │
│           the value will be dropped                │
│                                                     │
└─────────────────────────────────────────────────────┘
```

---

## 规则详解

```rust
fn main() {
    // 第一定律：每个值都有一个所有者
    let s = String::from("hello");  // s 是 "hello" 的所有者

    {
        // 第二定律：任一时刻只能有一个所有者
        let t = s;  // 所有权从 s 移动到 t
        // 现在 t 是所有者，s 不再有效

        println!("{}", t);  // ✅ 正确
        // println!("{}", s);  // ❌ 错误：s 已无效
    }
    // t 离开作用域

    // 第三定律：所有者离开作用域，值被丢弃
    let u = String::from("world");
}  // u 离开作用域，"world" 被自动释放
```

---

## 规则记忆口诀

```
一值一主，时刻唯一
主离作用域，值即丢弃
```
