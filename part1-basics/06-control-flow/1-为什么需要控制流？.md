## 6.1 为什么需要控制流？

### 没有控制流的程序

```
顺序执行：
┌─────────────────┐
│ 第 1 行代码       │
│ ↓               │
│ 第 2 行代码       │
│ ↓               │
│ 第 3 行代码       │
│ ↓               │
│ ...             │
│ ↓               │
│ 第 1000 行代码    │
└─────────────────┘

问题：无法根据条件做不同事情，无法重复执行
```

### 有控制流的程序

```
条件分支：
              ┌───────────────┐
              │   if 条件？    │
              └───────┬───────┘
                     │
         ┌───────────┴───────────┐
         │ YES                   │ NO
         ↓                       ↓
    ┌─────────┐            ┌─────────┐
    │ 执行 A   │            │ 执行 B   │
    └─────────┘            └─────────┘

循环重复：
              ┌───────────────┐
         ┌───│  条件满足？    │
         │   └───────┬───────┘
         │           │ YES
         │           ↓
         │      ┌─────────┐
         │      │ 执行代码 │
         │      └─────────┘
         │           │
         └───────────┘
```
