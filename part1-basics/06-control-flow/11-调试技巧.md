## 6.11 调试技巧

### 打印执行流程

```rust
fn main() {
    for i in 1..=5 {
        dbg!(i);  // 打印 i 的值和位置

        if i % 2 == 0 {
            println!("{} 是偶数", i);
        }
    }
}
```

### 条件断点

```rust
fn main() {
    let numbers = [1, 2, 3, 4, 5];

    for (i, &num) in numbers.iter().enumerate() {
        // 只在特定条件下打印
        if num == 3 {
            println!("找到 3 了！索引：{}", i);
        }
    }
}
```
