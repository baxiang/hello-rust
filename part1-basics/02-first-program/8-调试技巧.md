## 2.8 调试技巧

### 使用 dbg! 宏

```rust
fn main() {
    let x = 10;
    let y = 20;

    // dbg! 打印表达式和结果
    let sum = dbg!(x + y);

    // 输出：
    // [src/main.rs:5] x + y = 30

    // 可以链式使用
    let result = dbg!(dbg!(x * 2) + dbg!(y * 3));
}
```

### 使用 cargo expand

安装：
```bash
cargo install cargo-expand
```

查看宏展开：
```bash
cargo expand
```
