## 2.7 常见错误

### 错误 1：修改变量忘记 mut

```rust
fn main() {
    let x = 10;
    // x = 20;  // ❌ 错误

    let mut y = 10;  // ✅ 正确
    y = 20;
}
```

错误信息：
```
error[E0384]: cannot assign twice to immutable variable `x`
```

### 错误 2：变量未初始化就使用

```rust
fn main() {
    let x: i32;
    // println!("{}", x);  // ❌ 错误：未使用
}
```

### 错误 3：类型推断失败

```rust
fn main() {
    // let x;  // ❌ 无法推断类型
    // x = 10;

    let x: i32;  // ✅ 显式标注
    x = 10;
}
```
