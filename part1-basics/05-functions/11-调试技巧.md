## 5.11 调试技巧

### 使用 dbg! 调试函数

```rust
fn calculate(x: i32, y: i32) -> i32 {
    let sum = dbg!(x + y);
    let product = dbg!(x * y);
    let result = sum + product;
    dbg!(result)
}

fn main() {
    let r = calculate(3, 5);
    println!("最终结果：{}", r);
}
```

**输出：**
```
[src/main.rs:3] x + y = 8
[src/main.rs:4] x * y = 15
[src/main.rs:5] result = 23
最终结果：23
```

### 打印函数签名

```rust
fn my_function(x: i32, y: String) -> (i32, String) {
    (x, y)
}

fn main() {
    // 打印类型信息
    println!("函数类型：{:?}", std::any::type_name_of_val(&my_function));
}
```
