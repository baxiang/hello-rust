## 1.7 常见问题

### Q1: 下载慢怎么办？

配置国内镜像（以中科大为例）：

```bash
# 创建/编辑配置文件
mkdir -p ~/.cargo
cat > ~/.cargo/config.toml << 'EOF'
[source.crates-io]
replace-with = 'ustc'

[source.ustc]
registry = "sparse+https://mirrors.ustc.edu.cn/crates.io-index/"
EOF
```

### Q2: 编译错误：找不到 rustc

确保已加载环境变量：

```bash
source "$HOME/.cargo/env"
```

或者重启终端。

### Q3: 如何卸载 Rust？

```bash
rustup self uninstall
```
