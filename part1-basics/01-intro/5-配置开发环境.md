## 1.5 配置开发环境

### VS Code 配置（推荐）

1. **安装扩展**
   - 打开 VS Code
   - 按 `Ctrl+Shift+X`（或 `Cmd+Shift+X`）
   - 搜索并安装：
     - `rust-analyzer`（必须）
     - `Better TOML`（推荐）
     - `CodeLLDB`（调试用）

2. **rust-analyzer 功能**
   - 代码补全
   - 类型提示
   - 错误检查
   - 跳转到定义
   - 重命名

### 其他编辑器

| 编辑器 | 插件 |
|--------|------|
| **IntelliJ IDEA** | Rust 插件 |
| **Vim/Neovim** | rust.vim + rust-analyzer |
| **Emacs** | rust-mode + lsp-mode |
| **Sublime Text** | RustEnhanced |
