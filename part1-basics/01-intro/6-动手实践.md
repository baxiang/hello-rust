## 1.6 动手实践

### 练习 1：修改程序

编辑 `src/main.rs`：

```rust
fn main() {
    println!("你好，Rust!");
    println!("这是第二行");
    println!("第三行...");
}
```

运行：

```bash
cargo run
```

### 练习 2：查看编译输出

```bash
# 查看详细编译过程
cargo build -v

# 查看生成的二进制文件
file target/debug/hello_rust

# 查看文件大小
ls -lh target/debug/hello_rust
```

### 练习 3：创建库项目

```bash
# 创建库项目
cargo new my_lib --lib
cd my_lib
ls -la
```

库项目结构：

```
my_lib/
├── Cargo.toml
└── src/
    └── lib.rs    # 注意：是 lib.rs 不是 main.rs
```

编辑 `src/lib.rs`：

```rust
pub fn add(a: i32, b: i32) -> i32 {
    a + b
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_add() {
        assert_eq!(add(2, 3), 5);
    }
}
```

运行测试：

```bash
cargo test
```
