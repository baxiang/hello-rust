## 03.2 可变性

### mut 关键字

```rust
fn main() {
    // 使用 mut 声明可变变量
    let mut y = 5;
    println!("y = {}", y);

    y = 10;  // ✅ 正确：y 是可变的
    println!("y = {}", y);

    // 不可变 vs 可变
    let a = 5;      // 不可变
    let mut b = 5;  // 可变

    // a = 6;       // ❌ 错误
    b = 10;         // ✅ 正确
}
```

### 为什么默认不可变

```
┌─────────────────────────────────────────────────────┐
│          为什么变量默认不可变？                      │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. 安全性                                           │
│     • 防止意外修改                                   │
│     • 减少 bug                                       │
│                                                     │
│  2. 并发安全                                         │
│     • 不可变数据天然线程安全                        │
│     • 编译器可以做得更好优化                         │
│                                                     │
│  3. 代码可读性                                       │
│     •  reader 知道值不会改变                         │
│     • 更容易推理代码行为                             │
│                                                     │
│  Rust 哲学：让正确的事情变得容易                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```
