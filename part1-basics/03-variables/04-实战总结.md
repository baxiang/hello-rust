## 03.8 常见错误

### 错误 1：修改不可变变量

```rust
fn main() {
    let x = 5;
    // x = 6;  // ❌ 错误：不能修改不可变变量

    // ✅ 修复
    let mut x = 5;
    x = 6;
}
```

### 错误 2：未初始化使用

```rust
fn main() {
    // let x: i32;
    // println!("{}", x);  // ❌ 错误：使用未初始化变量

    // ✅ 修复
    let x: i32 = 0;
    println!("{}", x);
}
```

### 错误 3：常量缺少类型

```rust
// const MISSING_TYPE = 10;  // ❌ 错误：常量需要类型标注

// ✅ 修复
const HAS_TYPE: i32 = 10;
```

### 错误 4：可变静态变量不安全访问

```rust
static mut DATA: i32 = 0;

fn main() {
    // DATA = 1;  // ❌ 错误：需要 unsafe

    // ✅ 修复
    unsafe {
        DATA = 1;
    }
}
```
