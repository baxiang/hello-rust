## 03.6 类型推断

### 自动推断

```rust
fn main() {
    // Rust 可以推断类型
    let guess = 42;           // i32
    let pi = 3.14;            // f64
    let greeting = "Hello";   // &str
    let is_rust = true;       // bool
}
```

### 需要类型标注的情况

```rust
fn main() {
    // 1.  ambiguous 类型
    let num: i32 = "42".parse().unwrap();

    // 2. 默认类型不适用
    let big_num: u128 = 100;

    // 3. 空集合
    let mut vec: Vec<i32> = Vec::new();

    // 4. 函数参数和返回值
    fn add(x: i32, y: i32) -> i32 {
        x + y
    }
}
```

### 类型推断规则

```
┌─────────────────────────────────────────────────────┐
│              类型推断规则                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  整数默认为 i32                                      │
│  浮点数默认为 f64                                    │
│  布尔默认为 bool                                     │
│  字符串字面量为 &str                                 │
│                                                     │
│  推断流程：                                          │
│  1. 查看初始值                                       │
│  2. 查看使用上下文                                   │
│  3. 如果无法推断，需要显式标注                       │
│                                                     │
└─────────────────────────────────────────────────────┘
```
