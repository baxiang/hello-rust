## 03.4 静态变量

### static 关键字

```rust
// 静态变量使用 static 声明
static LANGUAGE: &str = "Rust";

// 可变静态变量（需要 unsafe）
static mut COUNTER: u32 = 0;

fn main() {
    println!("LANGUAGE = {}", LANGUAGE);

    // 访问可变静态变量需要 unsafe
    unsafe {
        COUNTER += 1;
        println!("COUNTER = {}", COUNTER);
    }
}
```

### static vs const

```rust
// const：每次使用都内联
const MAX: u32 = 100;

// static：有固定内存地址
static MIN: u32 = 0;

fn main() {
    // const 每次使用都是新值
    let a = MAX;
    let b = MAX;

    // static 有固定地址
    let ptr = &MIN as *const u32;
    println!("static 地址：{:p}", ptr);
}
```
