| 宏 | 用途 |
|------|------|
| `stringify!` | 转 token 为字符串 |
| `concat!` | 拼接字符串 |
| `env!` | 获取环境变量 |
| `include!` | 包含文件 |
| `compile_error!` | 编译错误 |
| `line!` / `file!` | 行号/文件名 |
| `module_path!` | 模块路径 |
| `format_args!` | 格式化参数 |
