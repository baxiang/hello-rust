## 25.9 调试宏

### expand! 宏

```rust
// 使用 cargo expand 查看宏展开
cargo install cargo-expand
cargo expand

// 查看特定文件的宏展开
cargo expand my_crate::main
```

### trace_macros!

```rust
// 启用宏追踪（需要 nightly）
#![feature(trace_macros)]

fn main() {
    trace_macros!(true);

    factorial!(5);

    trace_macros!(false);
}
```

### macro_rules! 调试技巧

```rust
macro_rules! debug_macro {
    ($($tt:tt)*) => {{
        // 打印原始 tokens
        println!("输入：{}", stringify!($($tt)*));

        // 继续正常处理
        // ...
    }};
}
```
