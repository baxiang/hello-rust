## 22.10 智能指针选择指南

```
┌─────────────────────────────────────────────────────┐
│              智能指针选择指南                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  需要堆上分配？                                      │
│  ├── 是，递归类型/大对象 → Box<T>                   │
│  └── 否 → 继续                                      │
│                                                     │
│  需要多线程共享？                                    │
│  ├── 是 → Arc<T>                                    │
│  │           ├── 需要修改？ → Arc<Mutex<T>>         │
│  │           └── 只读？ → Arc<T>                    │
│  └── 否 → 继续                                      │
│                                                     │
│  需要单线程共享？                                    │
│  ├── 是 → Rc<T>                                     │
│  │           ├── 需要修改？ → Rc<RefCell<T>>        │
│  │           └── 只读？ → Rc<T>                     │
│  └── 否 → Box<T> 或普通引用                         │
│                                                     │
│  需要打破循环引用？ → Weak<T>                       │
│                                                     │
└─────────────────────────────────────────────────────┘
```
