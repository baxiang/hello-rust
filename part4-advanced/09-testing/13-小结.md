### 测试命令速查

| 命令 | 说明 |
|------|------|
| `cargo test` | 运行所有测试 |
| `cargo test test_name` | 运行特定测试 |
| `cargo test -- --nocapture` | 显示输出 |
| `cargo test --doc` | 只运行文档测试 |
| `cargo doc --open` | 生成并打开文档 |
| `cargo llvm-cov` | 生成覆盖率报告 |
| `cargo bench` | 运行基准测试 |

### 测试金字塔

```
         /\
        /  \
       / E2E\      端到端测试（少量）
      /______\
     /        \
    /Integration\  集成测试（适量）
   /______________\
  /    Unit Tests   \  单元测试（大量）
 /____________________\
```

### 测试建议

1. **多写单元测试** - 快速、隔离、易维护
2. **适量集成测试** - 验证模块间交互
3. **文档测试** - 保证示例代码正确
4. **避免过度测试** - 测试实现细节会降低重构能力
