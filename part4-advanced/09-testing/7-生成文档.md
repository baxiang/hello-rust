## 28.7 生成文档

### Cargo doc

```bash
# 生成文档
cargo doc

# 生成并打开文档
cargo doc --open

# 包含私有项
cargo doc --private

# 不包含依赖文档
cargo doc --no-deps
```

### 文档属性

```rust
//! # My Crate
//!
//! 这是一个示例 crate
//!
//! ## 功能
//!
//! - 功能 1
//! - 功能 2
//!
//! ## 示例
//!
//! ```
//! use my_crate::add;
//! let result = add(2, 3);
//! assert_eq!(result, 5);
//! ```

/// 函数文档
#[doc(hidden)]  // 不显示在文档中
pub fn hidden_func() {}

/// 结构体文档
#[derive(Debug)]
pub struct MyStruct {
    /// 字段文档
    pub field: i32,
}

/// Trait 文档
pub trait MyTrait {
    /// 方法文档
    fn method(&self);
}
```
