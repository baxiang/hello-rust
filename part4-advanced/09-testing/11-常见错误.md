## 28.11 常见错误

### 错误 1：忘记 #[cfg(test)]

```rust
// ❌ 错误：测试代码会被编译到生产代码
mod tests {
    // ...
}

// ✅ 正确
#[cfg(test)]
mod tests {
    // ...
}
```

### 错误 2：测试依赖外部状态

```rust
// ❌ 不好
#[test]
fn test_with_global_state() {
    // 依赖全局变量或时间
}

// ✅ 好
#[test]
fn test_isolated() {
    // 自包含的测试，不依赖外部状态
}
```

### 错误 3：测试过于复杂

```rust
// ❌ 不好
#[test]
fn test_everything() {
    // 100 行测试代码，测试多个功能
}

// ✅ 好
#[test]
fn test_single_feature() {
    // 只测试一个功能
}
```
