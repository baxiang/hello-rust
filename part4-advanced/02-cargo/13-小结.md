### 常用命令速查

| 命令 | 说明 |
|------|------|
| `cargo build` | 构建项目 |
| `cargo build --release` | 发布构建 |
| `cargo run` | 运行项目 |
| `cargo test` | 运行测试 |
| `cargo check` | 快速检查 |
| `cargo fmt` | 格式化代码 |
| `cargo clippy` | 代码 lint |
| `cargo doc --open` | 生成文档 |
| `cargo add` | 添加依赖 |
| `cargo update` | 更新依赖 |
| `cargo tree` | 查看依赖树 |
