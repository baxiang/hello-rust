## 21.10 配置优化

### 配置镜像（国内用户）

```toml
# ~/.cargo/config.toml
[source.crates-io]
replace-with = 'ustc'

[source.ustc]
registry = "sparse+https://mirrors.ustc.edu.cn/crates.io-index/"

# 或者使用 rsproxy
[source.rsproxy]
registry = "sparse+https://rsproxy.cn/crates.io-index/"
```

### 编译加速

```toml
# ~/.cargo/config.toml
# 使用 sccache 缓存
[build]
rustc-wrapper = "sccache"

# 或者使用 ccache
rustc-wrapper = "ccache"
```

```bash
# 安装 sccache
cargo install sccache

# 启用
export RUSTC_WRAPPER=sccache
```

### 增加并行编译

```toml
# ~/.cargo/config.toml
[build]
jobs = 4  # 并行任务数
```
