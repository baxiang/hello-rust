## 20.9 常见错误

### 错误 1：模块文件找不到

```
error[E0583]: file not found for module `xyz`
 --> src/lib.rs:1:1
  |
1 | mod xyz;
  | ^^^^^^^^

原因：缺少 src/xyz.rs 或 src/xyz/mod.rs
解决：创建对应的模块文件
```

### 错误 2：私有模块无法访问

```rust
mod private {
    pub fn func() {}
}

fn main() {
    // private::func();  // ❌ 错误：模块是私有的
}

// 修复
pub mod public {
    pub fn func() {}
}

fn main() {
    public::func();  // ✅
}
```

### 错误 3：路径错误

```rust
// ❌ 错误
// use non_existent::module;

// ✅ 正确
use crate::existent::module;
```

### 错误 4：循环依赖

```rust
// ❌ 错误
mod a {
    use crate::b;  // a 使用 b
}

mod b {
    use crate::a;  // b 使用 a，循环依赖
}

// 解决：重构代码，避免循环依赖
```
