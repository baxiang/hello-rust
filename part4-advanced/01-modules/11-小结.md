### 模块系统速查表

| 语法 | 说明 |
|------|------|
| `mod foo;` | 声明模块 |
| `pub mod foo;` | 声明公共模块 |
| `use foo::bar;` | 引入模块 |
| `pub use foo::bar;` | 重新导出 |
| `crate::` | crate 根 |
| `super::` | 父模块 |
| `self::` | 当前模块 |
