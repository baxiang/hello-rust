# 项目一：待办事项 CLI

> 构建一个功能完整的命令行待办事项管理工具。

## 学习目标

完成本项目后，你将掌握：
- 使用 clap 处理命令行参数
- 设计模块化的项目结构
- 实现数据的持久化存储
- 处理用户输入和错误
- 发布可执行的 CLI 工具
