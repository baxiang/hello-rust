# 第 02 章：第一个 Rust 程序

> **本章代码基于 Rust 2024 Edition (Rust 1.85+) 编写**

> 深入理解 Hello World 程序，掌握 Rust 基本语法结构。

## 本章目标

完成本章学习后，你将掌握：
- 理解 Rust 程序的基本结构
- 掌握 `fn main()` 入口函数的作用
- 学会使用 `println!` 宏进行输出
- 了解变量声明和注释语法
- 掌握基本的调试技巧

## 核心概念速查

| 概念 | 说明 | 示例 |
|------|------|------|
| `fn main()` | 程序入口 | `fn main() { }` |
| `println!` | 打印宏（带换行） | `println!("Hello");` |
| `let` | 声明不可变变量 | `let x = 5;` |
| `let mut` | 声明可变变量 | `let mut x = 5;` |
| `const` | 常量 | `const MAX: i32 = 100;` |
| 遮蔽 | 重新使用变量名 | `let x = 1; let x = 2;` |
| `dbg!` | 调试宏 | `dbg!(x + y);` |

## 程序结构图解

```
┌─────────────────────────────────────────────────────────────┐
│                    Hello World 程序结构                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   fn main() {                                               │
│   │  │   │                                                  │
│   │  │   └── 花括号：函数体开始                              │
│   │  │                                                      │
│   │  └── main：入口函数名（必须是 main）                     │
│   │                                                         │
│   └── fn：函数关键字（function 的缩写）                     │
│                                                             │
│       println!("Hello, World!");                           │
│       │       │ │            │                              │
│       │       │ │            └── 分号：语句结束             │
│       │       │                                              │
│       │       │ └── 感叹号：这是宏，不是函数！               │
│       │                                                      │
│       │       └── 字符串：要输出的内容                       │
│       │                                                      │
│       └── println：打印并换行                                │
│                                                             │
│   }  ◄── 花括号：函数体结束                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 本地实验

本章示例代码位于 `examples/` 目录。

