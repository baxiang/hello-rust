## 使用建议

### 学习路径

1. **初学者路径**（1-2 月）
   - 第一部分 → 第二部分 → 第 14 章错误处理
   - 每章完成基础题 + 2 题进阶题

2. **进阶路径**（2-3 月）
   - 第三部分 → 第四部分（选择性）
   - 完成所有进阶题 + 1 题挑战题

3. **项目实战路径**
   - 选择感兴趣的章节
   - 重点完成挑战题
   - 结合 part5-projects 项目

### 答案验证

每道题后附有答案提示：
```rust

</details>
```

欢迎提交你的答案：
1. Fork 本仓库
