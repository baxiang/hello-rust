## Crate：最小的编译单元

### 什么是 Crate

```
┌─────────────────────────────────────────────────────┐
│              Crate 是什么                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Crate 是 Rust 最小的编译单元                        │
│                                                     │
│  两种类型：                                          │
│  ┌─────────────────────────────────────────────┐   │
│  │ 库 Crate (Library Crate)                    │   │
│  │  - 不能独立运行，提供可复用的代码            │   │
│  │  - 根文件：src/lib.rs                        │   │
│  └─────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────┐   │
│  │ 二进制 Crate (Binary Crate)                 │   │
│  │  - 可独立运行的程序                          │   │
│  │  - 根文件：src/main.rs                       │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Package：Crate 的容器

```bash
my_package/                  # 一个 Package
├── Cargo.toml               # Package 的清单文件
├── src/
│   ├── lib.rs               # 库 Crate（可选）
│   └── main.rs              # 二进制 Crate（可选）
└── tests/                   # 集成测试
```

一个 Package 可以包含：
- 至多一个库 Crate
- 多个二进制 Crate
- 但不能只有二进制 Crate 而没有库 Crate
