## 运行效果

```bash
# 添加待办事项
todo add "学习 Rust 基础"
todo add "完成所有权章节"
todo add "编写第一个项目"

# 列出待办事项
todo list

# 标记完成
todo done 1

# 列出所有（包括已完成）
todo list --all

# 删除待办事项
todo remove 2

# 清空已完成
todo clear
```
