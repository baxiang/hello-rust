## 第一步：创建项目

```bash
cargo new todo-cli
cd todo-cli
```

### Cargo.toml

```toml
[package]
name = "todo-cli"
version = "0.1.0"
edition = "2021"

[dependencies]
clap = { version = "4", features = ["derive"] }
serde = { version = "1", features = ["derive"] }
serde_json = "1"
anyhow = "1"
colored = "2"
```
