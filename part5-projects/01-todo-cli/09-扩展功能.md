## 扩展功能

### 添加优先级

```rust
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Todo {
    pub id: u32,
    pub content: String,
    pub done: bool,
    pub created_at: String,
    pub priority: Priority,  // 新增
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub enum Priority {
    Low,
    Medium,
    High,
}
```

### 添加标签

```rust
pub struct Todo {
    pub id: u32,
    pub content: String,
    pub done: bool,
    pub created_at: String,
    pub tags: Vec<String>,  // 新增
}
```

### 添加截止日期

```rust
pub struct Todo {
    pub id: u32,
    pub content: String,
    pub done: bool,
    pub created_at: String,
    pub due_date: Option<String>,  // 新增
}
```
