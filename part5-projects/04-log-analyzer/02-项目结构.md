## 项目结构

```
log-analyzer/
├── Cargo.toml
├── src/
│   ├── main.rs           # 入口
│   ├── lib.rs            # 库入口
│   ├── parser/
│   │   ├── mod.rs
│   │   ├── common.rs     # 通用日志格式
│   │   ├── nginx.rs      # Nginx 日志
│   │   └── json.rs       # JSON 日志
│   ├── analyzer/
│   │   ├── mod.rs
│   │   ├── stats.rs      # 统计分析
│   │   ├── filter.rs     # 过滤器
│   │   └── aggregator.rs # 聚合器
│   ├── report/
│   │   ├── mod.rs
│   │   ├── text.rs       # 文本报告
│   │   └── html.rs       # HTML 报告
│   └── monitor.rs        # 实时监控
└── tests/
    └── parser_test.rs
```
