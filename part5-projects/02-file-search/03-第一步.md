## 第一步：创建项目

```bash
cargo new minigrep
cd minigrep
```

### Cargo.toml

```toml
[package]
name = "minigrep"
version = "0.1.0"
edition = "2021"

[dependencies]
clap = { version = "4", features = ["derive"] }
regex = "1"
walkdir = "2"
colored = "2"
anyhow = "1"
ignore = "0.4"  # 支持 .gitignore

[dev-dependencies]
tempfile = "3"
```
