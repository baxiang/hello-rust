## 第一步：创建项目

```bash
cargo new image-processor
cd image-processor
```

### Cargo.toml

```toml
[package]
name = "image-processor"
version = "0.1.0"
edition = "2021"

[dependencies]
clap = { version = "4", features = ["derive"] }
image = "0.24"
tokio = { version = "1", features = ["full"] }
rayon = "1"
anyhow = "1"
tracing = "0.1"
tracing-subscriber = "0.3"
walkdir = "2"
indicatif = "0.17"  # 进度条
```
