## 项目结构

```
image-processor/
├── Cargo.toml
├── src/
│   ├── main.rs           # 入口
│   ├── lib.rs            # 库入口
│   ├── processor/
│   │   ├── mod.rs
│   │   ├── resize.rs     # 调整大小
│   │   ├── convert.rs    # 格式转换
│   │   ├── compress.rs   # 压缩
│   │   └── filter.rs     # 滤镜
│   ├── batch/
│   │   ├── mod.rs
│   │   └── worker.rs     # 批量处理
│   └── utils/
│       ├── mod.rs
│       └── format.rs     # 格式工具
└── tests/
    └── processor_test.rs
```
