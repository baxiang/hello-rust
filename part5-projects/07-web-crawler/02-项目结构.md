## 项目结构

```
web-crawler/
├── Cargo.toml
├── src/
│   ├── main.rs           # 入口
│   ├── lib.rs            # 库入口
│   ├── crawler/
│   │   ├── mod.rs
│   │   ├── fetcher.rs    # 网页获取
│   │   ├── parser.rs     # HTML 解析
│   │   ├── scheduler.rs  # 任务调度
│   │   └── queue.rs      # URL 队列
│   ├── extractor/
│   │   ├── mod.rs
│   │   ├── links.rs      # 链接提取
│   │   ├── text.rs       # 文本提取
│   │   ├── images.rs     # 图片提取
│   └── storage/
│       ├── mod.rs
│       └── file.rs       # 文件存储
└── config.toml           # 配置文件
```
