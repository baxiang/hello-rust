## 第一步：创建项目

```bash
cargo new mini-interpreter
cd mini-interpreter
```

### Cargo.toml

```toml
[package]
name = "mini-interpreter"
version = "0.1.0"
edition = "2021"

[dependencies]
anyhow = "1"
thiserror = "1"
rustyline = "13"  # REPL 支持

[dev-dependencies]
pretty_assertions = "1"
```
